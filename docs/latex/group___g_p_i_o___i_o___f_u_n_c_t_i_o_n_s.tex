Reads the current state of a GPIO pin. Pin should be configured as input or in a state where reading is desired (can read output pins too).

{\bfseries{Read Operations:\+}}
\begin{DoxyItemize}
\item Reads IDR (Input Data Register)
\item Returns bit value for specified pin
\item For output pins:\+ returns ODR value (what was written)
\end{DoxyItemize}

Sets or clears a GPIO output pin. Very fast operation (\texorpdfstring{$<$}{<} 1 µs). Pin must be configured as output before calling this function.

{\bfseries{Pin State Control:\+}}
\begin{DoxyItemize}
\item GPIO\+\_\+\+PIN\+\_\+\+SET (1):\+ Output HIGH (3.\+3V for STM32\+F756\+ZG)
\item GPIO\+\_\+\+PIN\+\_\+\+RESET (0):\+ Output LOW (0V)
\end{DoxyItemize}

{\bfseries{Register Operations:\+}}
\begin{DoxyItemize}
\item SET:\+ Write to BSRR (Bit Set/\+\+Reset Register) lower 16 bits
\item RESET:\+ Write to BSRR upper 16 bits (acts as clear)
\end{DoxyItemize}
