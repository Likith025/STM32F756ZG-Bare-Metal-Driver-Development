Configures EXTI (External Interrupt) for the specified GPIO pin. Enables falling edge detection (button press). The function:\+
\begin{DoxyEnumerate}
\item Selects GPIO port in SYSCFG\+\_\+\+EXTICR
\item Enables falling edge trigger in EXTI\+\_\+\+FTSR
\item Enables interrupt mask in EXTI\+\_\+\+IMR
\end{DoxyEnumerate}

{\bfseries{EXTI Configuration:\+}}
\begin{DoxyItemize}
\item Trigger:\+ Falling edge (signal goes from HIGH to LOW)
\item Interrupt:\+ Enabled and unmasked
\item Pre-\/condition:\+ \doxylink{group___g_p_i_o___i_n_i_t___f_u_n_c_t_i_o_n_s_gae0b97c0917307b554927fade1d4d17a0}{GPIO\+\_\+\+Init()} must be called first
\end{DoxyItemize}

{\bfseries{EXTI Register Mapping:\+}}
\begin{DoxyItemize}
\item EXTI0-\/4:\+ Individual interrupt vectors
\item EXTI5-\/9:\+ Shared vector EXTI9\+\_\+5\+\_\+\+IRQHandler
\item EXTI10-\/15:\+ Shared vector EXTI15\+\_\+10\+\_\+\+IRQHandler
\end{DoxyItemize}
