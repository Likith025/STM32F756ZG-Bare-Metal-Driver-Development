Main interrupt handler that processes all USART interrupt sources:\+
\begin{DoxyItemize}
\item TXEIE:\+ Transmit register empty → send next byte
\item TCIE:\+ Transmission complete → signal application
\item RXNEIE:\+ Receive register not empty → read byte
\item ORE:\+ Overrun error → signal error
\end{DoxyItemize}

{\bfseries{Interrupt Priority (Order of Checking):\+}}
\begin{DoxyEnumerate}
\item TXEIE + TXE:\+ Send next byte if available
\item TCIE + TC:\+ Signal TX complete, disable interrupts
\item RXNEIE + RXNE:\+ Receive next byte
\item RXNEIE + ORE:\+ Handle overrun error
\end{DoxyEnumerate}

{\bfseries{Word Length and Parity Handling:\+}}
\begin{DoxyItemize}
\item 8-\/bit no parity:\+ Full 8 bits data
\item 8-\/bit with parity:\+ 7 bits data + parity
\item 9-\/bit no parity:\+ Full 9 bits data
\item 9-\/bit with parity:\+ 8 bits data + parity
\end{DoxyItemize}

Initiates non-\/blocking interrupt-\/driven transmission. Function sets up the transmit buffer, enables TXEIE and TCIE interrupts, then returns immediately. \doxylink{group___u_s_a_r_t___i_n_t_e_r_r_u_p_t___f_u_n_c_t_i_o_n_s_ga19440941cb2860dc318565918c44e000}{USART\+\_\+\+IRQHandler()} automatically sends data in the background, calling \doxylink{stm32f7xx__uart__driver_8h_aa4732538a8f3f2d83fc5da935f36eb50}{USART\+\_\+\+Application\+Event\+Callback()} when transmission is complete.

{\bfseries{Operation:\+}}
\begin{DoxyEnumerate}
\item Check if TX not already busy (USART\+\_\+\+BUSY\+\_\+\+IT\+\_\+\+TX)
\item Store TX buffer pointer and length
\item Set TX state to busy
\item Enable TXEIE interrupt (transmit register empty)
\item Enable TCIE interrupt (transmission complete)
\item Return immediately
\end{DoxyEnumerate}
