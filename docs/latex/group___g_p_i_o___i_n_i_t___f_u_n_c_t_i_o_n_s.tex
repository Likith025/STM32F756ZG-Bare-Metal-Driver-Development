Enables the APB2 clock for the specified GPIO port. This must be called before configuring any pins in that port. The function:\+
\begin{DoxyEnumerate}
\item Identifies GPIO port
\item Sets corresponding RCC bit (AHB1\+ENR for GPIOx)
\item Waits for clock to stabilize
\end{DoxyEnumerate}

{\bfseries{RCC GPIO Enable Bits:\+}}
\begin{DoxyItemize}
\item GPIOA:\+ RCC\+\_\+\+AHB1\+ENR bit 0
\item GPIOB:\+ RCC\+\_\+\+AHB1\+ENR bit 1
\item GPIOC:\+ RCC\+\_\+\+AHB1\+ENR bit 2
\item GPIOD:\+ RCC\+\_\+\+AHB1\+ENR bit 3
\item etc.
\end{DoxyItemize}

{\bfseries{Clock Frequency:\+}}
\begin{DoxyItemize}
\item AHB1 Clock:\+ 216 MHz (System clock domain)
\item GPIO operates at full AHB1 frequency
\item No prescaling on GPIO port clock
\end{DoxyItemize}

Configures a GPIO pin with specified mode and speed. Must be called before using the pin. This function:\+
\begin{DoxyEnumerate}
\item Enables GPIO port clock
\item Sets pin mode (MODER register)
\item Sets output speed (OSPEEDR register)
\item Sets pull-\/up/\+pull-\/down (PUPDR register)
\item Sets output type (OTYPER register)
\end{DoxyEnumerate}
