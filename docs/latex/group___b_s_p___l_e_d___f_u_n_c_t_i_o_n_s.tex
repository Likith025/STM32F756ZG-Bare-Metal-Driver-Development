Initializes GPIO pin 15 on PORT B as an output for the blue LED indicator. Configuration identical to red LED but on different pin.

{\bfseries{GPIO Configuration:\+}}
\begin{DoxyItemize}
\item Port:\+ GPIOB
\item Pin:\+ 15
\item Mode:\+ Output (GPIO\+\_\+\+MODE\+\_\+\+OUT)
\item Speed:\+ High (for LED response visibility)
\item Pull:\+ None (open output)
\item Initial State:\+ Low (LED off)
\end{DoxyItemize}

{\bfseries{Pin and Port Details:\+}}
\begin{DoxyItemize}
\item Pin 15:\+ Adjacent to red LED pin 14
\item Same port allows simultaneous control
\item Can be used for status indication
\item Recommended:\+ Red=Error, Blue=Ready
\end{DoxyItemize}
