Initialize timer with specified configuration
\begin{DoxyParams}[1]{Parameters}

Initializes a timer peripheral with the configuration stored in the handler structure. Sets up prescaler, period, and counting direction. Enables the timer clock automatically.

{\bfseries{Initialization Steps:\+}}
\begin{DoxyEnumerate}
\item Enable timer clock via RCC
\item Set prescaler value (TIM\+\_\+\+PSC)
\item Set auto-\/reload value /\+ period (TIM\+\_\+\+ARR)
\item Configure counting direction (up/\+down)
\item Clear counter and flags
\item Timer is disabled; use \doxylink{group___t_i_m_e_r___f_u_n_c_t_i_o_n_s_ga39c741de08b18d032760b29b0763ec46}{Timer\+Control()} to start
\end{DoxyEnumerate}

{\bfseries{Register Mapping:\+}}
\begin{DoxyItemize}
\item TIM\+\_\+\+PSC:\+ Prescaler value (divides input clock)
\item TIM\+\_\+\+ARR:\+ Auto-\/reload register (period -\/ 1)
\item TIM\+\_\+\+CR1:\+ Control register (direction, enable bits)
\item TIM\+\_\+\+CNT:\+ Counter value (current count)
\end{DoxyItemize}


\begin{DoxyParams}[1]{Parameters}
\doxymbox{\texttt{in}}  & {\em timer\+\_\+handle} & Handler with configuration:\+
\begin{DoxyItemize}
\item p\+Timer:\+ Base address of timer (TIMER2, TIMER3, etc.)
\item Timer\+Config.\+Pre\+Scaler:\+ Clock divider value (1-\/65536)
\item Timer\+Config.\+Period:\+ Reload value (1-\/65535)
\item Timer\+Config.\+direction:\+ Up\+Counter (0) or Down\+Counter (1)
\end{DoxyItemize}\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Timer clock must be enabled in RCC before calling this function
\item Handler must be properly initialized with configuration values
\item After initialization, timer is stopped; call Timer\+Control(handle, ENABLE) to start
\item Prescaler acts as (PSC + 1) divisor
\item Period acts as (ARR + 1) reload count
\end{DoxyItemize}
\end{DoxyNote}
\begin{DoxyWarning}{Warning}

\begin{DoxyItemize}
\item Zero prescaler/\+period causes undefined behavior
\item Invalid timer address causes register corruption
\item Do not call while timer is running
\end{DoxyItemize}
\end{DoxyWarning}
