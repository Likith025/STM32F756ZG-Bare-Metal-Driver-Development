Clear a USART flag via ICR (Interrupt Clear Register).

Clears a USART flag by writing to the Interrupt Clear Register (ICR). Flags must be cleared after processing to allow them to be set again. Only specific flags can be cleared via ICR; others are cleared by reading/\+writing data.

{\bfseries{Clearable Flags (via ICR write):\+}}
\begin{DoxyItemize}
\item USART\+\_\+\+ICR\+\_\+\+PECF:\+ Parity error clear flag
\item USART\+\_\+\+ICR\+\_\+\+FECF:\+ Framing error clear flag
\item USART\+\_\+\+ICR\+\_\+\+NECF:\+ Noise flag clear flag
\item USART\+\_\+\+ICR\+\_\+\+ORECF:\+ Overrun error clear flag
\item USART\+\_\+\+ICR\+\_\+\+IDLECF:\+ IDLE line clear flag
\item USART\+\_\+\+ICR\+\_\+\+TCCF:\+ Transmission complete clear flag
\end{DoxyItemize}

{\bfseries{Auto-\/\+Clearing Flags:\+}}
\begin{DoxyItemize}
\item TXE:\+ Cleared by writing to TDR
\item RXNE:\+ Cleared by reading RDR
\item TC:\+ Can be cleared by writing to ICR or reading ISR+writing TDR
\end{DoxyItemize}

{\bfseries{ICR Register Operation:\+}} Writing 1 to ICR bit clears corresponding ISR flag. Writing 0 has no effect (safe). Multiple flags can be cleared with single write.
