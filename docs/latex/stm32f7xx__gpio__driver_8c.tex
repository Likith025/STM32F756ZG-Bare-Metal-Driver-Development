\doxysubsection{Detailed Description}
GPIO (General Purpose Input/\+\+Output) Driver for STM32\+F756\+ZG. 

\begin{DoxyAuthor}{Author}
Likith025 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
April 2026 
\end{DoxyDate}
This module provides GPIO port and pin management functions for the STM32\+F756\+ZG microcontroller. It includes functions for:\+
\begin{DoxyItemize}
\item GPIO port clock enable/\+disable
\item GPIO pin mode configuration (input, output, alternate function)
\item GPIO output operations (set/\+clear/\+toggle pins)
\item GPIO input operations (read pin state)
\item External interrupt (EXTI) configuration
\item GPIO interrupt flag management
\end{DoxyItemize}

{\bfseries{GPIO Features:\+}}
\begin{DoxyItemize}
\item 16 pins per port (Port A-\/I available)
\item Configurable I/\+O modes (input, output, analog, alternate function)
\item Flexible speed settings (low, medium, high, very high)
\item Optional internal pull-\/up/\+pull-\/down resistors
\item Open drain or push-\/pull output
\item External interrupt capability on all pins
\end{DoxyItemize}

{\bfseries{Port Mapping:\+}}
\begin{DoxyItemize}
\item GPIOA:\+ Base address 0x40020000, used for buttons, timers
\item GPIOB:\+ Base address 0x40020400, used for LEDs
\item GPIOC:\+ Base address 0x40020800, used for UART
\item GPIOD-\/I:\+ Additional ports for expansion
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item GPIO clocks must be enabled in RCC before calling these functions
\item EXTI configuration requires SYSCFG clock enable
\item Some pins have limited alternate function options
\item Pin state changes are immediate (no queuing)
\end{DoxyItemize}
\end{DoxyNote}
\begin{DoxyAttention}{Attention}

\begin{DoxyItemize}
\item Invalid port addresses cause system hang or hard fault
\item Do not access pins before GPIO clock enable
\item EXTI handlers must be implemented by application
\item Interrupt flags must be cleared in handler
\end{DoxyItemize}
\end{DoxyAttention}
