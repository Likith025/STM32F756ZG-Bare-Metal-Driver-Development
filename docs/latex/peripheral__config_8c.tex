\doxysubsection{Detailed Description}
Board Support Package (BSP) Configuration for STM32\+F756\+ZG. 

\begin{DoxyAuthor}{Author}
Likith025 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
April 2026 
\end{DoxyDate}
This module contains hardware initialization functions for all peripherals used in the embedded CLI application. It provides simple, high-\/level setup functions that configure:\+
\begin{DoxyItemize}
\item GPIO for LED indicators (red, blue on PORT B)
\item GPIO for button input (PORT A)
\item USART3 for serial communication (TX on C10, RX on C11)
\item TIMER2 for PWM generation (CH1 on A0)
\end{DoxyItemize}
