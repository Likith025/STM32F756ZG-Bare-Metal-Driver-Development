Wrapper function that configures both the priority and enable state of an interrupt. This is the typical entry point for interrupt configuration in applications.

{\bfseries{Priority Values:\+}}
\begin{DoxyItemize}
\item 0:\+ Highest priority (executed immediately)
\item 1-\/14:\+ Intermediate priorities
\item 15:\+ Lowest priority (executed after higher priority interrupts)
\end{DoxyItemize}

\begin{DoxyItemize}
\item 0-\/15:\+ System exceptions (handled by Cortex-\/\+M7)
\item 16+:\+ Peripheral interrupts (USART, GPIO, TIMER, etc.)
