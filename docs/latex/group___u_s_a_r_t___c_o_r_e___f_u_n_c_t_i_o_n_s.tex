Enables or disables the USART peripheral clock via RCC APB1\+ENR register. The clock must be enabled before any USART configuration or operation.

{\bfseries{Clock Mapping (APB1\+ENR):\+}}
\begin{DoxyItemize}
\item USART2:\+ Bit 17 (offset from USART2\+\_\+\+BASE:\+ 0×400 → position 0)
\item USART3:\+ Bit 18 (offset from USART2\+\_\+\+BASE:\+ 0×400 → position 1)
\item UART4:\+ Bit 19 (offset from USART2\+\_\+\+BASE:\+ 0×800 → position 2)
\item UART5:\+ Bit 20 (offset from USART2\+\_\+\+BASE:\+ 0x\+C00 → position 3)
\end{DoxyItemize}
