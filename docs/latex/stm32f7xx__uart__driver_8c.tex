\doxysubsection{Detailed Description}
STM32\+F7xx USART Driver -\/ Serial Communication Interface. 

\begin{DoxyAuthor}{Author}
Likith025 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
February 28, 2026 
\end{DoxyDate}
This module provides comprehensive USART (Universal Synchronous/\+\+Asynchronous Receiver/\+\+Transmitter) driver functionality for STM32\+F756\+ZG microcontroller. It supports both blocking and interrupt-\/driven data transmission and reception with flexible configuration options.

{\bfseries{Supported Features:\+}}
\begin{DoxyItemize}
\item Configurable baud rates (300 to 4,500,000 bps)
\item Word lengths:\+ 8-\/bit or 9-\/bit
\item Stop bits:\+ 1, 1.\+5, or 2 bits
\item Parity:\+ None, Even, or Odd
\item Oversampling:\+ 8x or 16x (affects baud rate precision)
\item Hardware flow control:\+ None, CTS only, RTS only, CTS+\+RTS
\item Transmission modes:\+ Blocking, Interrupt-\/driven
\item Interrupt-\/driven RX with error detection
\item Automatic overflow error handling
\end{DoxyItemize}

{\bfseries{USART Instances:\+}}
\begin{DoxyItemize}
\item USART2:\+ Base 0x40004400, IRQ 38, APB1 clock (42 MHz)
\item USART3:\+ Base 0x40004800, IRQ 39, APB1 clock (42 MHz)
\item UART4:\+ Base 0x40004\+C00, IRQ 52, APB1 clock (42 MHz)
\item UART5:\+ Base 0x40005000, IRQ 53, APB1 clock (42 MHz)
\item USART1:\+ Base 0x40011000, IRQ 37, APB2 clock (84 MHz)
\item USART6:\+ Base 0x40011400, IRQ 71, APB2 clock (84 MHz)
\end{DoxyItemize}

{\bfseries{Data Flow (Interrupt-\/\+Driven):\+}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{TX:\ Application\ →\ pTXBuffer\ →\ TXEIE\ ISR\ →\ TDR\ →\ Serial\ Line}
\DoxyCodeLine{RX:\ Serial\ Line\ →\ RDR\ →\ RXNEIE\ ISR\ →\ pRXBuffer\ →\ Application}

\end{DoxyCode}


{\bfseries{Register Architecture:\+}}
\begin{DoxyItemize}
\item CR1:\+ Control register 1 (enable, TX/\+\+RX, interrupts, word length, parity)
\item CR2:\+ Control register 2 (stop bits, line break control)
\item CR3:\+ Control register 3 (hardware flow control, DMA)
\item BRR:\+ Baud rate register (mantissa and fraction)
\item TDR:\+ Transmit data register
\item RDR:\+ Receive data register
\item ISR:\+ Interrupt and status register (flags)
\item ICR:\+ Interrupt clear register (flag clear)
\end{DoxyItemize}

{\bfseries{Clock Configuration:\+}}
\begin{DoxyItemize}
\item APB1:\+ 42 MHz (USART2-\/5, UART4-\/5)
\item APB2:\+ 84 MHz (USART1, USART6)
\item Define USART\+\_\+\+CLK in header for calculations
\end{DoxyItemize}

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item GPIO alternate functions must be configured before USART use
\item Interrupt handlers must be linked in startup code
\item Application must implement \doxylink{stm32f7xx__uart__driver_8h_aa4732538a8f3f2d83fc5da935f36eb50}{USART\+\_\+\+Application\+Event\+Callback()}
\item Baud rate error \texorpdfstring{$<$}{<} 3\% for reliable operation
\end{DoxyItemize}
\end{DoxyNote}
\begin{DoxyAttention}{Attention}

\begin{DoxyItemize}
\item Ensure GPIO pins are configured as alternate functions
\item USART clock must be enabled via RCC before use
\item Interrupt handler must clear flags to prevent stalling
\item TX/\+\+RX buffers must remain valid during transfer
\end{DoxyItemize}
\end{DoxyAttention}
