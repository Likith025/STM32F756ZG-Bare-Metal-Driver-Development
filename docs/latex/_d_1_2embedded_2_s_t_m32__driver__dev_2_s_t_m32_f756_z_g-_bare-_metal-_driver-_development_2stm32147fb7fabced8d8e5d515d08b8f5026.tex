Initializes GPIO pin 14 on PORT B as an output for the red LED indicator. Sets up push-\/pull output with high speed capability for LED visibility.

{\bfseries{GPIO Configuration:\+}}
\begin{DoxyItemize}
\item Port:\+ GPIOB
\item Pin:\+ 14
\item Mode:\+ Output (GPIO\+\_\+\+MODE\+\_\+\+OUT)
\item Speed:\+ High (for LED response visibility)
\item Pull:\+ None (open output)
\item Initial State:\+ Low (LED off)
\item Push-\/\+Pull:\+ Yes (standard LED drive)
\end{DoxyItemize}

{\bfseries{Register Setup:\+}}
\begin{DoxyItemize}
\item GPIOB\+\_\+\+MODER\+[28:\+29]\+ = 01 (Output mode)
\item GPIOB\+\_\+\+OSPEEDR\+[28:\+29]\+ = 10 (High speed)
\item GPIOB\+\_\+\+OTYPER\+[14]\+ = 0 (Push-\/pull)
\item GPIOB\+\_\+\+PUPDR\+[28:\+29]\+ = 00 (No pull)
\item GPIOB\+\_\+\+ODR\+[14]\+ = 0 (Initial low -\/ LED off)
\end{DoxyItemize}

{\bfseries{LED Drive Current:\+}}
\begin{DoxyItemize}
\item Output HIGH:\+ \texorpdfstring{$\sim$}{\string~}100 mA (GPIO max, limited by external resistor)
\item Forward Voltage:\+ 2.\+0-\/3.\+0 V typical for red LED
\item Typical Series Resistor:\+ 100-\/200 Î©
\item Power Consumption:\+ \texorpdfstring{$\sim$}{\string~}10-\/20 mW per LED
\end{DoxyItemize}
